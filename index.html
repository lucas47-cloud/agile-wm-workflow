⚖️ Modus-C-Übergabe-Prompt für Juri

Scoring zuerst befüllen (Schritt 2) und auf „Weiter zu Juri" klicken – dann erscheint hier der fertige Modus-C-Prompt.
→ Juri in ChatGPT öffnen ↗ Prompt kopieren → in Juri einfügen → Compliance-Score in Scoring-Tabelle eintragen
⚠️ Hinweis: Juri ist kein Anwalt und gibt keine Rechtsberatung. Die Analyse fasst zusammen was in den ToS steht. Für verbindliche Aussagen wende dich an einen Fachanwalt.

Drei Ausbaustufen – von heute bis zur vollautomatisierten Pipeline.

Stufe 1 · Jetzt
Hybrid-Modus
Manuelle Eingabe + Paste-Back. Scoring und Juri-Prompt automatisch.
  • 1Formular ausfüllenManuell
  • 2Prompt generieren → Claude öffnet sich1 Klick
  • 3WM-Antwort einfügen → JSON auto-erkanntPaste
  • 4Scoring + Juri-Prompt erscheinen automatischAuto
  • 5Juri-Prompt in ChatGPT einfügenManuell

Stufe 2 · Nächster Schritt
API-Integration
Claude API direkt im Browser. Kein Copy-Paste – Scoring nach einem Klick.
  • 1Formular ausfüllenManuell
  • 2„Analyse starten" klicken1 Klick
  • 3WM-API gibt Analyse + Scoring zurückAuto
  • 4Scoring Matrix befüllt sich sofortAuto
  • 5Juri-Prompt in ChatGPT einfügenManuell
Manuelle Aktionen: 2 · Voraussetzung: Claude API-Key · Kosten: ~0,005 € / Analyse
Stufe 3 · Vollautomatisierung
Vollständige Pipeline
WM + Juri + Report vollautomatisch. Ein Klick – vollständiger Entscheidungsreport.
  • 1Formular ausfüllenManuell
  • 2„Analyse starten" klicken1 Klick
  • 3WM analysiert → Scoring automatischAuto
  • 4Juri analysiert ToS → Compliance-ScoreAuto
  • 5Entscheidungsreport als PDF exportierbarAuto
Manuelle Aktionen: 1 · Voraussetzung: Backend + Juri-API · Zeithorizont: 2–3 Monate

🗺️ Was für Stufe 3 noch gebraucht wird

🔌Juri als API
Custom GPT braucht eine eigene Schnittstelle oder wird als eigenständiges Projekt mit programmatischem Zugriff umgebaut
🖥️Kleines Backend
Node.js oder Python-Server der beide API-Calls orchestriert und den Report zusammenführt – kein großes Projekt
📄Report-Template
Definiertes Dokument-Format (PDF/Word) in das WM-Analyse, Scoring und Juri-Output automatisch eingefügt werden
🔒Auth & Zugriffsschutz
Login oder Token-Schutz wenn die App öffentlich erreichbar sein soll – verhindert API-Missbrauch
